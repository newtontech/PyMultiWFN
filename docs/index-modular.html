Quick Start

Installation

Install PyMultiWFN with a single command:

pip install pymultiwfn

Basic Usage

import pymultiwfn as pw

# Load wavefunction
wf = pw.Wavefunction.from_file('molecule.fchk')

# Calculate electron density
density = wf.calculate_density(grid)

# AI-enhanced analysis
cps = wf.find_critical_points_with_ai(grid, density)
