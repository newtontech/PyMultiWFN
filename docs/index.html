Installation Guide

Quick Installation

Get PyMultiWFN up and running on your system with pip:

# Install the latest stable version
pip install pymultiwfn

# Install with optional dependencies for GPU support
pip install pymultiwfn[gpu]

# Install development version
pip install pymultiwfn[dev]

System Requirements

  • Python 3.8+
  • NumPy 1.20+
  • SciPy 1.7+
  • 4GB RAM minimum (8GB+ recommended)

Verification

After installation, verify PyMultiWFN is working:

python -c "import pymultiwfn as pw; print(pw.__version__)"

API Reference

Core Classes

Wavefunction Class

The main class for representing quantum chemical wavefunctions:

import pymultiwfn as pw

# Load from file
wf = pw.Wavefunction.from_file('molecule.fchk')

# Calculate electron density
density = wf.calculate_density(grid)

Grid Class

Handles spatial grid generation:

# Generate uniform grid
grid = pw.Grid.uniform(
    center=[0, 0, 0],
    size=[10, 10, 10],
    spacing=0.1
)
